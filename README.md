# Log Sensitivity Analyzer
Bu proje, sistem ve uygulama logları içerisindeki hassas verileri (şifre, IP, API anahtarı vb.) tespit etmek için geliştirilmiş bir güvenlik aracıdır.

## Kurulum ve Çalıştırma
1. Python'ın yüklü olduğundan emin olun.
2. Projeyi indirin.
3. Terminal üzerinden: `python src/main.py` komutunu çalıştırın.
